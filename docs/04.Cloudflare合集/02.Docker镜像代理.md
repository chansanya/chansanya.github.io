## 使用


### Docker镜像：
```
docker pull proxy.lhyan.cn/user/image
```

### ghcr.io 镜像：
```
docker pull proxy.lhyan.cn/ghcr.io/user/image
```

### Quay.io 镜像：
```
docker pull proxy.lhyan.cn/quay.io/org/image
```

###  K8s 镜像：
```
docker pull proxy.lhyan.cn/registry.k8s.io/pause:3.8
```


### 一键设置镜像加速
修改文件 `/etc/docker/daemon.json`（如果不存在则创建）

### 创建目标路径
```shell
sudo mkdir -p /etc/docker
```

### 新建daemon.json文件
```shell
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["https://proxy.lhyan.cn"]
}
EOF
```

### 重新加载并重启
```shell
sudo systemctl daemon-reload&&systemctl restart docker
```
