## Linux


### 创建存放路径
```shell
mkdir  /opt/java/
```

下载对应版本的jdk包（略）

### 修改配置文件

```shell
vi /etc/profile
```

在最底部插入如下内容
```shell
export JAVA_HOME=/opt/java/jdk1.8.0_171
export CLASSPATH=$CLASSPATH:$JAVA_HOME/lib/
export PATH=$PATH:$JAVA_HOME/bin
```

### 加载配置

```shell
source /etc/profile 
```
