关于注释

注释打的好，代码容易找

# see

/**
 * 注释
 * @see  String
 * @see  String#value
 * @see  String#toString() 
 */

点击分别可跳转到

  • String类
  • String 的value 属性
  • String 的toString 方法

注意

@see 只能作用在多行注释每行的开始位置

# link

可用 {@link } 达到上述效果

/**
 * 注释
 * {@link String}  
 * {@link String#value}  
 * {@link String#toString()}  
 */

注意

{@link} 能作用在任意位置

示例 作用
{@link 类名} 跳转类
{@link 类名#方法名} 跳转方法
{@link 类名#属性名} 跳转属性
{@link 方法名} 跳转到本类的方法
{@link 属性名} 跳转到本类的属性
{@link } 跳转到网址。
{@link <a href="https://knife.blog.csdn.net/">https://knife.blog.csdn.net</a>} 跳转到网址。
