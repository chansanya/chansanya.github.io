VsCode插件

# Auto Close Tag

自动闭合 HTML/XML 标签的插件。当你输入开始标签的 > 时，自动补全对应的闭合标签，支持 HTML、Vue、JSX 等文件格式。

核心功能：

  • 自动添加闭合标签
  • 自动重命名配对标签
  • 支持自定义触发字符

# Jupyter

在 VSCode 中直接运行和编辑 Jupyter Notebook 文件（.ipynb）。提供交互式编程体验，适合数据分析、机器学习等场景。

核心功能：

  • 原生支持 .ipynb 文件
  • 单元格级别的代码执行
  • 内联显示图表和输出结果
  • 支持多种编程语言内核（Python、Julia、R 等）

# Material Icon Theme

为 VSCode 提供 Material Design 风格的文件和文件夹图标主题。通过直观的图标快速识别不同类型的文件。

核心功能：

  • 丰富的文件类型图标（支持数百种文件格式）
  • 文件夹图标自动匹配常见项目结构
  • 高度可定制的图标配置
  • 支持自定义文件夹颜色

# Git Graph

可视化 Git 提交历史的插件。以图形化方式展示分支、合并、提交记录，让 Git 操作更直观。

核心功能：

  • 图形化展示 Git 提交树
  • 查看提交详情和文件变更
  • 直接在图形界面执行 Git 操作（checkout、merge、rebase 等）
  • 支持多仓库管理

# CSS Peek

快速查看和跳转到 CSS 定义的插件。在 HTML 或 JSX 文件中，可以直接预览或跳转到对应的 CSS 样式定义。

核心功能：

  • 悬停预览 CSS 定义
  • 快速跳转到样式文件
  • 支持 CSS、SCSS、LESS 等预处理器
  • 适用于 HTML、Vue、React 等框架

# Claude Code

Anthropic 官方推出的 AI 编程助手插件。基于 Claude 大模型，提供智能代码补全、重构、调试等功能。

核心功能：

  • AI 驱动的代码生成和补全
  • 智能代码重构和优化建议
  • 自然语言交互式编程
  • 支持多种编程语言和框架
  • 上下文感知的代码理解

# Trailing Spaces

高亮显示和删除代码中的尾随空格。帮助保持代码整洁，避免不必要的空白字符提交到版本控制。

核心功能：

  • 高亮显示行尾空格
  • 一键删除所有尾随空格
  • 保存时自动删除（可配置）
  • 支持自定义高亮颜色

# Vue Peek

专为 Vue 开发设计的插件。快速查看和跳转到 Vue 组件定义，提升 Vue 项目的开发效率。

核心功能：

  • 快速跳转到 Vue 组件定义
  • 悬停预览组件内容
  • 支持 Vue 2 和 Vue 3
  • 识别组件的 props、methods、computed 等
