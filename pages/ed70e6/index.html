Nginx处理跨域

使用 Nginx 解决跨域问题配置

     location / {
        add_header Access-Control-Allow-Origin 'https://domain.cn';
        add_header Access-Control-Allow-Methods '*';
        add_header Access-Control-Allow-Headers '*';
        # 预检请求       
        if ($request_method = 'OPTIONS') {
                return 204;
        }

        proxy_pass  http://localhost;
    }
